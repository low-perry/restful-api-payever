# Project overview

This is a restful api build with the Nestjs framework.
Due to time related issues the test suites are not fully complete.
In order for the app to work fully, please provide your own `MONGO_DB_URI`.



## Installation

```bash
$ npm install
```

## Running the app

```bash

# watch mode
$ npm run start:dev

## Test

```bash
# unit tests
$ npm run test
